###### Requirements

2. Root (Magisk 23.0+ / Kitsune Mask)
3. 64-bit CPU (preferred)

###### Description

**Stryker App [MOD]** is a custom modification and revival of the legendary mobile pentesting application. While the original project was discontinued by its developer ZalexDev, this version has been completely refactored to function fully autonomously in offline mode without relying on official backend servers.

It allows you to test your networks and devices for common vulnerabilities without specialized skills, while providing an easy way to inject your own exploits and custom modules.

---

### 🚀 What's New in the MOD Version?

* 🌐 **Full Offline Operation:** Completely removed dependencies on the dead official servers. The app no longer deadlocks or freezes while trying to fetch updates or news logs.
* 📦 **Module Integration:** Bundled with automated scripts to handle sub-system deployment and exploit asset unpacking.
* 🛠️ **Automation Batch Scripts:** Added root-level helper scripts (`build.bat`, `install.bat`, `log.bat`) to streamline the development and deployment workflow.

---

### 📦 Repository Structure

* `Stryker.4.0R/` — The core modification folder containing all reconstructed app resources and assets.
* `build.bat` — Automates the compilation and building process of the project.
* `install.bat` — Instantly pushes and installs the compiled build onto your device via ADB.
* `log.bat` — Streams filtered Logcat outputs for convenient real-time debugging.
